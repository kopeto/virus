## Installation
Install Boost-random1.65:
```bash
sudo apt-get install libboost-random1.65-dev
```

Install SDL2:
```bash
sudo apt-get install libsdl2-dev libsdl2-image-dev libsdl2-ttf-dev
```

Install FFmpeg:
```bash
sudo apt-get install ffmpeg
```

Build:
```bash
make
```

Run
```bash
./virus initial-values
```
